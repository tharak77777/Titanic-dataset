Titanic Dataset Analysis:

This project analyzes the Titanic dataset to identify factors affecting passenger survival.

Objectives:
- Clean and preprocess data
- Analyze survival rates by gender
- Analyze survival rates by passenger class
- Analyze survival rates by age groups
- Create visualizations using Seaborn and Matplotlib

Technologies Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

Result:
- Female passengers had higher survival rates.
- First-class passengers had better chances of survival.
- Children were more likely to survive than adults.
- Passenger age distribution showed most travelers were young adults.
